ENCRYPTER

C#

This project was something I had in my mind for quite sometime, but this project kept getting pushed. But, finally during the 2020 quarantine I decided and told myself, that I should stop pushing it further.
ENCRYPTER, takes any text from you and a password, and via using an algorithm that I developed, it encypts the text. To decrypt it the same password is needed. There is no way to recover the text if in case one forgets the password. There is no global or master key which can be used to decrypt the text.
This project was developed in Visual Studio 2019, using C#.
